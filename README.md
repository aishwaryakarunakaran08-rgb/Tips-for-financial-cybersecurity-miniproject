# Tips for Financial Cybersecurity Mini-Project

A clean, responsive, and modern single-page web application designed to educate users about account safety, how account takeovers happen, and essential practices to safeguard personal and financial information online.

## 🚀 Features

- **Educational Hero Sections:** Visually clear summaries describing Customer Account Takeovers.
- **Threat Analysis Grid:** Interactive-feeling cards breakdown vectors like Phishing, Smishing, Fake Websites, Social Engineering, and Stolen Credentials.
- **Risk Indicators:** A structured list displaying the warning signs of account compromise with priority risk badges (`High Risk`, `Warning`).
- **8 Essential Security Practices:** A custom-styled structural grid containing actionable tips to protect banking details.
- **Quick Security Checklist:** A checklist component utilizing inline SVGs for instant scannability.
- **Clean Responsive Design:** Crafted with a modern dark blue aesthetic using fluid layouts (`CSS Grid`, `Flexbox`) suitable for both desktop and mobile viewports.

## 🛠️ Tech Stack

- **HTML5:** Semantic architecture (`<header>`, `<section>`, `<footer>`, SVG icons).
- **CSS3:** Custom responsive layout grids, hover animations, color themes, and typography settings.
- **Google Fonts:** Utilizing `Playfair Display` for editorial headers and `Inter` for highly legible body prose.
